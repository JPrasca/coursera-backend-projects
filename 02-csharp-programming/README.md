# LibraryManager - Programación en C#

## 📖 Descripción del Proyecto
Este proyecto es una aplicación de consola desarrollada en C# que simula un sistema de gestión de biblioteca (`LibraryManager`). Su objetivo principal es poner en práctica los conceptos fundamentales de la **Programación Orientada a Objetos (POO)**.

## 🚀 Características
- Gestión de inventario de libros utilizando clases y objetos.
- Uso de colecciones (`List`, `Dictionary`) para almacenar datos en memoria.
- Implementación de principios de encapsulamiento.
- Menú interactivo en consola para realizar operaciones básicas (Agregar, Buscar, Listar).

## 🛠️ Tecnologías
- **Lenguaje:** C#
- **Framework:** .NET 8.0 (o superior)

## ⚙️ Cómo ejecutar el proyecto
1. Abre tu terminal y navega hasta esta carpeta:
   ```bash
   cd 02-csharp-programming
   ```
2. Ejecuta la aplicación con el siguiente comando:
   ```bash
   dotnet run
   ```
